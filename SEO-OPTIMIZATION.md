## Дополнительные рекомендации для улучшения SEO

### 1. Яндекс.Вебмастер
1. Зарегистрируйтесь в [Яндекс.Вебмастер](https://webmaster.yandex.ru/)
2. Добавьте свой сайт
3. Получите код верификации и добавьте его в `app/layout.tsx`:
   ```typescript
   verification: {
     yandex: 'ВАШ_КОД_ВЕРИФИКАЦИИ',
   }
   ```
4. После верификации отправьте sitemap: https://stroiremont156rf.ru/sitemap.xml

### 2. Замените URL сайта
В следующих файлах замените `https://stroiremont156rf.ru` на ваш реальный домен:
- `app/layout.tsx` (metadataBase)
- `app/page.tsx` (JSON-LD и OpenGraph)
- `app/sitemap.ts` (baseUrl)
- `app/robots.ts` (sitemap и host)
- `app/karkasnye-doma-pod-kluch/[slug]/layout.tsx` (OpenGraph)
- `app/karkasnye-doma-pod-kluch/[slug]/page.tsx` (JSON-LD)

### 3. Оптимизация изображений
- Убедитесь, что все изображения оптимизированы (WebP формат)
- Размер изображений не должен превышать 200-300 KB
- Используйте lazy loading для изображений ниже первого экрана

### 4. Скорость загрузки
Проверьте скорость сайта:
- [PageSpeed Insights](https://pagespeed.web.dev/)
- Настройте кэширование
- Минимизируйте CSS и JavaScript

### 5. Контент
- Добавьте больше уникального текста на страницы (минимум 500-1000 слов)
- Используйте ключевые слова естественно в тексте
- Добавьте заголовки H2, H3 с ключевыми словами

### 6. Внутренняя перелинковка
- Добавьте ссылки между связанными страницами
- Создайте хлебные крошки (breadcrumbs)
- Добавьте блок "Похожие дома" на всех страницах

### 7. Социальные сети
Добавьте ссылки на соцсети в `app/page.tsx` в массив `sameAs`:
```typescript
sameAs: [
  'https://vk.com/ваша-группа',
  'https://t.me/ваш-канал',
  // другие соцсети
]
```

### 8. Аналитика
Установите:
- Яндекс.Метрика
- Google Analytics (опционально)

Добавьте счетчики в `app/layout.tsx`

### 9. Микроразметка для отзывов
Добавьте Schema.org разметку Review для отзывов клиентов:
```json
{
  "@type": "Review",
  "author": "Имя клиента",
  "reviewRating": {
    "@type": "Rating",
    "ratingValue": "5"
  },
  "reviewBody": "Текст отзыва"
}
```

### 10. Локальный SEO
Добавьте LocalBusiness разметку для лучшего ранжирования в локальном поиске:
```json
{
  "@type": "LocalBusiness",
  "name": "Стройремонт 156",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "ул. Пушкина, д. 1",
    "addressLocality": "Оренбург",
    "postalCode": "460000",
    "addressCountry": "RU"
  },
  "telephone": "+7 (963) 180-10-34"
}
```

## Проверка индексации

После внесения изменений и деплоя:

1. Проверьте sitemap: `https://ваш-домен.ru/sitemap.xml`
2. Проверьте robots.txt: `https://ваш-домен.ru/robots.txt`
3. В Яндекс.Вебмастер проверьте:
   - Индексацию страниц
   - Ошибки сканирования
   - Структурированные данные
4. Используйте команду в Google: `site:ваш-домен.ru`
